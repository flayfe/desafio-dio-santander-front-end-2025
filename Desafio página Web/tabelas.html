Criando Tabelas com HTML

Anteriormente, as tabelas eram utilizadas para estruturar os sites, porém não usa-se mais por conta de três motivos principais:
Semântica - espera-se dados tabulares, mas acaba "enganando" os navegadores, leitores de tela e motores de busca
Acessibilidade - devido a dificuldade de interpretar os dados que estáo dentro da tag table, os leitores de tela acabam se perdendo
Tamanho do arquivo do HTML - a tabela acaba se tornando maior que as tags utilizadas hoje para estruturação (div e span), deixando o site mais lento e a manutenção do código mais complexa.
