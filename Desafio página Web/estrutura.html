Segue duas curiosidades sobre as tags que aprendi nas aulas:


Tag <font>

Embora esteja obsoleta (futuramente não vai mais receber suporte), a tag font permite alterar a aparência do texto na própria estrutura do HTML. Para melhorar a manutenção do código e a separação de funções entre HTML e CSS, esta tag foi depreciada.

Este texto foi alterado por uma tag font
Atributos aplicados: <font color="#03ab18" size="5" face="Algerian" font-style="i">



Tags <div e span>

As tags div e span geram diferenças na renderização do conteúdo da página. Isso ocorre porque a tag div é um elemento de bloco que ocupa todo o espaço vertical disponível independente do tamanho do seu conteúdo. Já a tag span é um elemento inline que se mantém no tamanho do seu conteúdo.

O abacaxi está dentro de uma div, fazendo o abacaxi ocupar uma linha inteira:

Abacaxi

A maça está dentro de uma span, fazendo a maça continuar sem alteração na exibição da página: Maça
