Entendendo HTML Semântico

Semântico diz respeito ao significado das palavras. Tendo isso como base, o HTML Semântico é quando usamos as tags que dizem o que elas significam ao invés de usar somente <div> ou <span> para tudo.

Dessa forma, fica mais fácil entender o código e o que cada parte da página faz. Ao tornar o código mais claro, os leitores de telas usados por pessoas com deficiência visual também conseguem entender melhor a estrutura da página e passam a informação mais compreensível.

Outro benefício é que com tudo mais organizado e nomeado de forma clara, os motores de busca entendem melhor o conteúdo e podem ranquear melhor o site.

Exemplo de HTML não Semântico 

                    
    <body>
        <div>...</div>
        <div>...</div>
        <div>...</div>
    </body>

Exemplo de HTML Semântico 

                    
    <body>
        <header>...</header>
        <main>...</main>
        <footer>...</footer>
    </body>
