Introdução ao HTML na Prática

O que aprendi nas aulas:
Neste módulo, o professor Diogo Mainardes compartilha diversas dicas legais para quem está começando com HTML. Ele mostra algumas ferramentas utilizadas no desenvolvimento como o Visual Studio Code junto com seus plugins favoritos que facilitam o trabalho.

Embora algumas dicas sejam óbvias para alguns, são essenciais para facilitar os estudos. Convenções, boas práticas, estruturação básica e recomendada, diferença entre tags e a melhor aplicação para cada uma. O Diogo explica tudo tendo ciência que os alunos podem ter diferentes níveis de conhecimento, por isso mostra desde elementos realmente básicos para quem está começando, como também complementa com informações mais avançadas, como por exemplo a escolha das tags que serão usadas pensando também na acessibilidade da aplicação.

Gostei bastante da aula em que ele mostra como utilizar o inspetor de elementos e ensina algumas interações e jeitos de ver os dados.
