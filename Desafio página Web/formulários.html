Trabalhando com Formulários em HTML

Existem muitas tags voltadas para formulários facilitando muito na hora de criar interfaces completas para entrada de dados. Achei a tag input super coringa por permitir, através dos seus atributos, adaptar para diversos tipos de campos. Um exemplo é criar um campo para o usuário digitar senha, mantendo a senha escondida com pontos com o atributo password. Também é possível criar um botão com diversas funcionalidades com esta tag, deixando a semântica um pouco de lado.

A explicação dos atributos checkbox e radio foi tão rica, que até possíveis problemas foram antecipados na video aula. Um dos problemas é, ao utilizar o tipo radio, caso não informe um name ou não coloque o mesmo name para as opções da questão, o navegador vai manter os valores clicados selecionados, sendo que a característica desse atributo é selecionar uma OU outra opção.
